## 概述

New API 是一个新一代大模型网关与 AI 资产管理系统，基于 [One API](https://github.com/songquanpeng/one-api) 进行二次开发。该项目提供了一个统一的接口来管理和使用各种 AI 模型服务。

<Callout type="info">
  New API 无缝集成了主要 AI 平台，包括 OpenAI、Anthropic、Google
  Gemini、Midjourney 等众多服务。
</Callout>

### 核心特性

- **统一接口** - 多个 AI 提供商的单一 API 端点
- **安全管理** - 先进的身份验证和授权
- **使用分析** - 全面的追踪和报告
- **高性能** - 为速度和可靠性进行了优化
- **多租户支持** - 完美适配团队和企业
