    HTTPS should be used in production environments to ensure authentication tokens. HTTP is only recommended for development environments.

    A layered log query system that supports administrators viewing site-wide logs and users viewing personal logs. It provides real-time statistics (RPM/TPM), multi-dimensional filtering, historical data cleanup, and other features. The Token query interface with CORS support facilitates third-party integration.

- **HTTP Method**: GET
- **Path**: `/api/log/self/stat`
- **Authentication Requirement**: User
- **Feature Description**: Retrieve log statistics for the current user, including quota consumption, request frequency, and Token usage
